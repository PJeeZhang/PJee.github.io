本地私有库创建

1：创建远程Spec文件仓库 获取远程地址
eg：https://git.dev.tencent.com/husbandbig/SpecDepository.git
2：创建本地的私有库索引文件夹，并与远程私有索引库进行关联
命令：    
    pod repo            // (查看本地已存在的索引库)     
    pod repo add + 本地私有索引库文件夹名称 + 远程Spec文件仓库地址        //添加本地私有索引库，并与远程私有索引库想关联
    eg:
        pod repo add SpecDepository https://git.dev.tencent.com/husbandbig/SpecDepository.git
3：创建远程组件代码仓库 获取远程地址
eg: https://git.dev.tencent.com/husbandbig/PJTestComponent.git
拉取代码  
创建模板        
命令：pod lib create  + 名称       //eg：pod lib create PJTestComponent
把代码放到Classes文件夹下面并且替换ReplaceMe.m文件
4：将组件代码提交到远程仓库
命令：      
    git status  
    git add .  
    git commit -m '基础组件测试工程'  
    编辑.podspec文件  
    pod lib lint    //验证  （pod lib lint --allow-warnings 忽略警告）  
    git status  
    git add .  
    git commit -m '编辑spec文件'  
    git remote add origin + 远程仓库组件的地址  
    git push origin master  
    git tag '0.1.0'                     //  (要与.podspec文件中的tag值保持一致 即s.version)  
    git push --tags           //(将tag提交到远程)  
    pod spec lint  //验证 （pod spec lint --allow-warnings 忽略警告）
5：将.podspec文件提交到本地私有索引库中
pod repo push +本地私有索引库文件名称 + 组件.podspec    //（pod repo push +本地私有索引库文件名称 + 组件.podspec --allow-warnings 忽略警告）这个同时也会提交到远程索引库中
6：结束 在其他项目的Podfile文件顶部添加 索引库的远程地址 source ‘远程索引库地址’ 然后正常pod 就行了
